## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/HornTech.git
```

Navigate to the project folder:

```bash
cd HornTech
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
